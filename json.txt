JSON - JavaScript Object Notation

복잡한 정보의 구조를 단순화시켜서 구조적으로 표현할 수 있는 데이터 타입

이름 : 김철수
나이 : 27
주소 : 부산광역시
키 : 165.7cm
초등학교 : A초등학교
중학교 : B중학교
고등학교 : C고등학교

{
	key: value
	"name": "김철수:,
	"age": 27,
	"addr": "부산광역시",
	"height": 165.7,
	"school": ["A초등학교", "B중학교", "C고등학교"]
}

key는 영어로
value(값) 자리에 올 수 있는 데이터의 형태
 - 문자, 문자열
 - 정수, 실수
 - 배열
 - JSON(객체)
